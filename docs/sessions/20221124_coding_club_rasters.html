No problem Big rasters? No problem! From [`raster` documentation](https://rspatial.github.io/raster/reference/raster-package.html): _The package can work with large files because the objects it creates from these files only contain information about the structure of the data, such as the number of rows and columns, the spatial extent, and the filename, but it does not attempt to read all the cell values in memory. In computations with these objects, data is processed in chunks._ This holds true for `terra` and `stars` as well, of course. --- background-image: url({{ site.baseurl}}/assets/images/background_challenge_2.png) class: left, top # Challenge 2 1.
