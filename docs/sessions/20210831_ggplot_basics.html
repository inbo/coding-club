Aesthetics This works: ``` ggplot(ToothGrowth, aes(x = dose, y = len, color = supp)) + geom_violin() ``` This works too: ``` ggplot(ToothGrowth) + geom_violin(aes(x = dose, y = len, color = supp)) ``` What should I use? Both are good, but...
